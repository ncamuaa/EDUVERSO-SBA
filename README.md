📌 Project Overview

EduVerso is an integrated digital learning platform designed to enhance student engagement and academic performance. The system consists of:

📱 Student Mobile Application (Flutter)
🌐 Admin Web Dashboard
⚙️ Backend API (Node.js + MySQL)

The platform provides AI-assisted learning, module access, peer feedback, and real-time announcements managed by administrators.

🎯 Objectives
To provide an accessible mobile learning platform for students
To integrate AI-based tutoring for enhanced understanding
To centralize academic modules, feedback, and announcements
To allow administrators to manage all system content efficiently
